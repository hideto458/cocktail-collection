# Cocktail Collection

## 概要

「TheCocktailDB」からランダムでドリンクメニューを取得するアプリ

## 機能

- ランダムにドリンクを取得し、画像とレシピを表示する機能
- ブックマーク機能（ローカルストレージに保存）
